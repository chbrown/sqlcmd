# sqlcmd

Alpha stability and tuned for PostgreSQL, via `pq` from [node-postgres](https://github.com/brianc/node-postgres).


## Installation

**From npm:**

```sh
npm install sqlcmd
```

**Or github:**

```sh
git clone https://github.com/chbrown/sqlcmd.git
cd sqlcmd
npm install -g
```

**Import:**

```js
var sqlcmd = require('sqlcmd');
```
